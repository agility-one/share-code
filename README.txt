Mise en place :
----------------

1) Créer un Dossier "content" à la racine du projet pour stocker les fichiers texte.

2) Mettre en place les fichiers PHP selon la structure donnée. 
.
├── content
│   └── <fichiers .txt générés>
├── cleanup.php
├── index.php
├── page.php
├── style.css
└── submit_content.php

3) penser à changer l'url dans submit_content.php pour votre url de site

4) Configurez un Cron Job pour exécuter cleanup.php toutes les 24 heures pour supprimer les .txt ou mode manuel (au choix)

Et partager !
